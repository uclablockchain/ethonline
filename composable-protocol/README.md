# composable

## Install
 - Download & install
 - ```bash
   git clone
   https://github.com/blockchainatucla/ethonline
   cd ethonline
   npm install
   ```
 - Save your testnet mnemonic and infura key into `.env`:
 - ```bash
   # contents of .env
   INFURA_KEY="abcdefghijklmnop"
   MNEMONIC="this is not actually a valid mnemonic but it should make sense"
   ```
